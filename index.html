KOMET

I have also made an AI - driven metro management system for Kochi metro as an Smart India Hackathon project. This model what basically does is that it takes some input files like Metro trains available, routes to be taken, adveritsement exposure hours required, no. of hours travelled by the train etc. Based on this information it tells the user/ system officials that which metro should go for repair, which should go route, which should stay back at the station. It basically improves the system efficiency and reduces the required labour to do all this work manuallly. Hence the name : KOMET is derived as "Kochi + Metro = KOMET".
